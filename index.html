アンケートビルダー 使い方ガイド

アンケートを作成してJSONBin.ioに回答データを保存し、はてなブログにiframeで埋め込むまでの手順を説明します。

Step 1 — JSONBin.ioの準備（無料）

1
アカウント作成
jsonbin.io にアクセスして無料アカウントを作成します。メールアドレスとパスワードだけで登録できます。
2
APIキーの取得
ログイン後、ダッシュボード右上のアイコンから「API Keys」を開き、「+ CREATE ACCESS KEY」でAPIキーを生成します。表示された $2a$10$... から始まる文字列をコピーします。
3
ビルダーにAPIキーを入力
左パネルの「JSONBin APIキー」欄にペーストします。Bin IDは空欄のままにしておくと、最初の回答時に自動で作成されます。
Bin IDを手動で指定することもできます。JSONBinのダッシュボードでBinを先に作成して、そのIDを入力してください。

Step 2 — アンケートの設定

1
基本情報を入力
「タイトル」と「説明文」を入力します。タイトルはアンケートページのページタイトルにもなります。
2
表示オプションを設定
  • 回答後に集計結果を表示：ON にすると送信後に棒グラフ形式の集計が表示されます
  • 重複回答チェック：ON にすると同じブラウザからの2回目以降の送信をブロックします（localStorageを使用）
  • 回答者数を表示する：ON にすると現在の回答件数を表示します。表示タイミングを「常時」「送信後のみ」「両方」から選べます
3
完了メッセージを設定
送信後に表示されるメッセージとサブメッセージを自由に入力できます。プレビューパネルの下部にリアルタイムで表示を確認できます。

Step 3 — デザインの選択

「デザインテーマ」から9種類のカラーテーマを選びます。テーマはブログのデザインに合わせて選んでください。

  • クリーン：白ベースのシンプルなデザイン
  • ミッドナイト：ダークテーマ（赤アクセント）
  • フォレスト・さくら・オーシャン・アンバー：各カラーの明るい系
  • ミニマル：モノクロ・余白重視
  • ラベンダー：紫系パステル
  • ペーパー：クラフト紙風

「選択肢のスタイル」から6種類の入力UIを選びます。

  • クラシック：標準のラジオボタン・チェックボックス
  • ピル型：タグのようにクリックして選択
  • ブロック型：ボタンと選択肢が一体化した横幅フル
  • カード型：アイコン付きのグリッドカード
  • トグル型：横並びのセグメント切り替え（単一選択向き）
  • アウトライン型：枠線のみ・選択時に強調

Step 4 — 質問の追加と編集

「＋ 質問を追加する」ボタンまたは右上の「+ 質問追加」ボタンで質問を追加します。追加後、自動的に編集パネルが開きます。

質問タイプ

  • 単一選択：選択肢から1つだけ選ぶ（ラジオボタン）
  • 複数選択：選択肢から複数選べる（チェックボックス）。「最低選択数」も設定可能
  • 自由記述：テキストエリアのみ。プレースホルダー文言を設定できます
  • 自由+選択肢：選択肢＋自由記述テキストエリアの複合タイプ

編集パネルの各設定

  • 選択肢：テキストを直接編集できます。「×」で削除、「＋ 追加」で選択肢を増やします
  • 最低選択数：複数選択・自由+選択肢タイプで設定可。0 = 制限なし
  • 必須：ON にすると未回答での送信をブロックします
  • ↑ / ↓：質問の順序を変更できます
プレビューパネルはリアルタイムで更新されます。右パネルでフォームの見た目を確認しながら編集してください。

Step 5 — エクスポートと公開

1
HTMLをダウンロード
右パネルの「エクスポート」タブを開き、「HTMLダウンロード」をクリックします。ファイル名はアンケートのタイトルになります。
2
GitHub Pagesに公開
GitHubで新しいリポジトリを作成し、ダウンロードしたHTMLをアップロードします。リポジトリの Settings → Pages → Branch を main に設定して公開します。公開URLは https://ユーザー名.github.io/リポジトリ名/ファイル名.html になります。
3
はてなブログに埋め込む
「埋め込みコード」タブのiframeコードをコピーし、src="" の値を実際の公開URLに書き換えてブログ記事のHTML編集欄に貼り付けます。
はてなブログでiframeを使うには「はてなPro」プランが必要です。また、同一ドメインでない外部iframeはセキュリティポリシーにより表示されない場合があります。その場合はアンケートページへのリンクを直接貼ることで代替できます。

よくある質問

  • 回答データはどこに保存されますか？
    JSONBin.ioのBinに保存されます。無料プランでは最大10,000件まで保存可能です。
  • APIキーが流出しても大丈夫ですか？
    JSONBinのAPIキーはHTMLソースに含まれるため、閲覧者がソースを見れば確認できます。書き込み専用キーを使うか、自前のバックエンド（GASなど）を挟むことを推奨します。
  • 重複回答チェックは完璧ですか？
    localStorageを使った簡易的なチェックのため、別のブラウザや別デバイスからの重複回答は防げません。本格的な制御にはログイン認証が必要です。
  • アンケートHTMLを修正したい
    ダウンロードしたHTMLをテキストエディタで直接編集できます。
